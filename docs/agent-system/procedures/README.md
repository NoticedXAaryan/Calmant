# Procedure Index

These procedures are operational runbooks. Agent skills can reference them, and implementation work should map product workflows to these procedures.

## Procedures

- `telegram-request-to-completion.md`: inbound Telegram message to completed run.
- `browser-task.md`: evidence-backed browser navigation and form automation.
- `approval-gates.md`: approval creation, resolution, and resume.
- `memory-write.md`: safe memory extraction and write policy.
- `presentation-creation.md`: end-to-end presentation generation.
- `job-application-goal.md`: long-term internship/job goal workflow.
- `release-verification.md`: release and regression checks.

## Rule

If a workflow is repeated twice, convert it into a skill document and add a replay/eval case.
