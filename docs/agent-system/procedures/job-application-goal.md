# Procedure: Job or Internship Application Goal

## Purpose

Run a long-term goal to find, prepare, and apply for opportunities.

## Steps

1. Create `Goal`.
2. Capture constraints:
   - role type,
   - location,
   - deadline,
   - eligibility,
   - preferred companies,
   - resume source,
   - portfolio links.
3. Schedule daily opportunity search.
4. Search sources.
5. Deduplicate opportunities.
6. Score fit.
7. Store opportunities.
8. Notify owner with shortlist.
9. For approved opportunities:
   - generate tailored CV,
   - draft cover email or answers,
   - prepare application.
10. Stop before submission.
11. Ask approval.
12. Submit after approval.
13. Store confirmation.
14. Monitor email for outcomes.
15. Schedule follow-up.

## Approval Required

- sending recruiter email,
- submitting application,
- uploading CV,
- claiming any qualification not already in profile,
- contacting third parties.

## Acceptance Criteria

- Daily search runs.
- Opportunities are tracked.
- CVs are tailored but truthful.
- No application submits without approval.
- Email outcomes update opportunity status.
